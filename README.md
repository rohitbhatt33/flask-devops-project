# 🚀 DevOps Flask Application Deployment

## 📌 Project Overview
This project demonstrates containerization and deployment of a Flask web application using Docker and AWS EC2 with Nginx reverse proxy.

## 🛠 Tech Stack
- Python (Flask)
- Docker
- Nginx
- AWS EC2 (Ubuntu)
- DockerHub

## ⚙️ Features
- Dynamic hostname display
- Environment-based versioning
- Health check endpoint (/health)
- Reverse proxy configuration

## 🚀 Deployment Steps
1. Build Docker image
2. Push to DockerHub
3. Deploy on EC2
4. Configure Nginx reverse proxy

## 🏆 Outcome
Production-style containerized deployment on AWS.
